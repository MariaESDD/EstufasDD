WEBSITE - ESTUFAS DOM DINIS

Como abrir:
1. Extraia o ficheiro ZIP.
2. Abra o ficheiro index.html no navegador.
3. O ficheiro style.css contém todo o design do site.
4. As fotografias estão na pasta assets/fotos.

Estrutura:
- index.html
- style.css
- assets/fotos/
- assets/favicon.svg

Tema:
Website elegante e simples sobre as estufas da Escola Secundária Dom Dinis, em Coimbra.
